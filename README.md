# Todo API
Task API, with user and status management for each task.
> Made with Python-DjangoRestFramework

## Technologies
* Python
* Django
* DjangoRestFramework
* Heroku
* Swagger UI Documentation

## Setup
Create a virtual environment

### On macOs and Linux
`python3 -m venv env`

### On macOs and Linux
`py -m venv env`
